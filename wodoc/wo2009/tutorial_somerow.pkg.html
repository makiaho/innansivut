Somerow

This tutorial explains how to make active record pattern based somerowXXX class.

Hannu Lohtander
hannu.lohtander@uta.fi

Active Record is Software Pattern that defines way to implement 4 basic SQL queries as a class. That class in wo2009 framework is SomeRow. Class is in library/some/database/row/row.php. ISomeRow.

SomeRow implements base methods for create, read, update and delete operations (CRUD). Programmer making new table to database, must create extending class and implement 3 required methods.

There is pdf explaining this process in details: http://www.cs.uta.fi/wo/wo2010/pub/pdf

Creating Active Record for table "demo" having column id and data is explained step by step.

  • Create new file "demo.php" to: library/some/database/rows/demo.php

  • create class SomeRowDemo extends SomeRow to file demo.php

  • Implement methods getColumns(), getTable() and getPrimary()

  • getColumns() must return column names.

  • getTable() must return table name

  • getPrimary() must return primary key name, the one with SERIAL datatype.

Usage Example: 

$demorow = SomeRow::getRow('demo');
$newid = $demorow->create();
$demorow->read();
$demorow->data = "new data";
$demorow->update();
$demorow->delete();
