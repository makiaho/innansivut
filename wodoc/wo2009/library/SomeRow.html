Class Details



[line 24]

SomeRow is factory class to create specialized SomeRow___ objects.

SomeRow povides getRow factory method. It gets arguments that is the suffix of the class extending SomeRow.  Extending class must be on subfolder rows/ in a file named exactly as class prefix. Example:  
  1.  get class SomeRowDemo from file rows/demo.php
    2. 

  2.  This is NOT singleton
    3. 

  3.  SomeRow::getRow('demo');
    4. 

Extendng class MUST implement getTable(), getPrimary() and getColumns methods. See methods for explanation.
